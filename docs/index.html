Integrated GitHub GUI
About this GUI
This GUI is a program utilizing Java Swing, a GitHub Subprocess Extension, and a GitHub API Extension.

Below are a list of Features:
  • Create Repository using GitHub username token
  • Will check if token is valid and if repository name is taken
  • Will git add changes to specified RepoPath
  • Ability to switch to a darker theme
Requirements in order to run:
  • Java version 8 or higher
